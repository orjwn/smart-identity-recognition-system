# Shared Utilities

This folder contains shared Python helpers used by the runtime, model wrappers, evaluation scripts and preparation tools.

- `helpers.py` contains alignment, SCRFD decode and visualisation helpers adapted from the attributed baseline where marked.
- `logging.py` configures local logging.
- `utils.py` contains additional shared helper functions.

Keep attribution comments in this folder because some helper blocks are copied or lightly modified from external sources.
